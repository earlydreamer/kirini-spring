1. 개요

60% 키보드는 일반 풀사이즈 키보드의 약 60%에 해당하는 크기로, 60~61개의 키만 포함한 컴팩트한 레이아웃입니다. 숫자 패드, 화살표 키, 기능 키(F1~F12), 그리고 홈/엔드/페이지업/페이지다운 등의 내비게이션 키가 생략되어 있습니다. 이러한 키들은 일반적으로 Fn 키와 다른 키의 조합으로 접근할 수 있습니다.

2. 특징

  • 알파벳 키, 숫자 키, 모디파이어 키(Shift, Ctrl, Alt 등)만 포함
  • 일반적으로 60~61개의 키로 구성
  • 화살표 키, 기능 키, 숫자 패드 등이 없음
  • Fn 키와의 조합을 통해 생략된 키 기능 사용
  • 가로 길이 약 29cm 내외의 컴팩트한 크기
[60% 키보드 레이아웃 이미지]

3. 장단점

장점

  • 책상 공간을 효율적으로 사용 가능
  • 휴대성이 우수함
  • 손의 움직임이 최소화되어 타이핑 효율 향상 가능
  • 마우스와의 거리가 가까워 인체공학적으로 유리
  • 커스터마이징 옵션이 다양함

단점

  • 화살표 키와 기능 키 사용 시 Fn 키 조합이 필요해 초기 적응이 어려움
  • 숫자 입력이 많은 작업에는 불편할 수 있음
  • 레이어(Layer) 시스템에 익숙해져야 함
  • 일부 게임이나 특수 소프트웨어에서 키 부족 문제 발생 가능

4. 대표 모델

  • Ducky One 2 Mini: 뛰어난 품질과 내구성, 다양한 스위치 옵션
  • GK61: 저렴한 가격, 핫스왑 소켓 지원

  • Leopold FC660M: 60%보다 약간 큰 65% 레이아웃, 뛰어난 품질
